# Ship_Docker_Simulation
Using Parallel Programing principles to simulate real-life scenarios.In this simulation a cargo ship docker port is being simulated were ships arrive in different ports based on their cargo category ,get assigned to a docking place and are unloaded through the help of cranes to load and unload crates and then transport the crates to the warehouse.
To run the demo of the project go to Ship_Docker_Simulation\dist\PortSimulation
